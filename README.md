## What Does This App Do?

This app helps teachers create **5 different versions** of the same lesson:

1. **Simplified** - Easier words for students who need support
2. **On-Level** - Just right for grade level
3. **Enriched** - More challenging for advanced learners
4. **Visual-Heavy** - Lots of pictures and diagrams
5. **Scaffolded** - Step-by-step instructions

All materials are generated by AI running **on your own computer** - no internet needed after setup!

## Quick Download Guide

### Step 1: Download This App

**Option A: Download ZIP (Easiest)**
1. Click the green **"Code"** button above
2. Click **"Download ZIP"**
3. Unzip the file and put the folder somewhere you'll remember

**Option B: Use Git**
```bash
git clone https://github.com/Tech-Inclusion-Pro/Local-Assignment-Differentiator-Application.git
```

### Step 2: Install Python

You need Python 3.10 or newer.

- **Check if you have it:** Open Terminal (Mac/Linux) or Command Prompt (Windows) and type:
  ```bash
  python3 --version
  ```
- **Don't have it?** Download from: https://www.python.org/downloads/

### Step 3: Install Ollama (The AI)

Ollama runs the AI on your computer.

1. Go to: https://ollama.ai
2. Click **Download** for your system
3. Install it and open it
4. Open Terminal/Command Prompt and run:
   ```bash
   ollama pull llama3.2
   ```

### Step 4: Run the App

**Mac/Linux:**
```bash
cd path/to/Local-Assignment-Differentiator-Application
chmod +x run.sh
./run.sh
```

**Windows:**
```cmd
cd path\to\Local-Assignment-Differentiator-Application
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

That's it! The app will open and you're ready to go!

## Features

### Wizard Mode
- **7-step guided process** to create your materials
- **Built-in Tutorial** button explains each step (5th grade reading level)
- **AI Chat** to refine your ideas before generating

### Dashboard
- **Save your assignments** for later use
- **Rename assignments** to keep organized
- **Delete assignments** you don't need
- **Add reflections** about what worked and what didn't
- **Export** to Word, PDF, PowerPoint, or Excel

### Navigation
- **Dashboard**, **Wizard**, **Settings**, **Reset**, and **Logout** buttons
- Active button stays highlighted so you always know where you are

## Using the App

### First Time Setup
1. Make sure Ollama is running (look for the llama icon)
2. Create an account when the app opens
3. Click the green **Tutorial** button to learn how to use the wizard

### Creating Materials
1. Click **Wizard** to start
2. Follow the 7 steps:
   - Step 1: What should students learn?
   - Step 2: What are your students' needs?
   - Step 3-5: Optional details to improve results
   - Step 6: Chat with AI for suggestions
   - Step 7: Generate and export!

### Managing Assignments
1. Click **Dashboard** to see saved assignments
2. Double-click an assignment to view it
3. Use **Rename** to change the name
4. Use **Delete** to remove it
5. Use **Load into Wizard** to edit and regenerate

## Troubleshooting

### "Cannot connect to Ollama"
- Make sure Ollama is running
- Open Terminal and type: `ollama serve`

### "No models installed"
- Run: `ollama pull llama3.2`

### App won't start
- Make sure Python 3.10+ is installed
- On Windows, make sure virtual environment is activated
- Try running `pip install -r requirements.txt` again

### Generation is slow
- First generation takes longer (AI loading into memory)
- Try a smaller model: `ollama pull llama3.1:8b`
- Close other programs to free memory
